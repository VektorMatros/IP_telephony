Команды USSD для операторов мобильной связи в Украине


Тут будут изложены все команды для стандартных запросов по Украинским операторам связи

KyivStar

Итак:

*111# - проверка счета
*112# - проверить бонусный счет по условиям тарифного плана
*112*1# - бонусы по заказанным пакетам
*112*3# - бонусы СМС
*252# - бонусные мегобайты из заказанных пакетов услуг
*161# - проверка номера телефона SIM карты
*107*1# - активая основной SIM
*107*2# - активация второстепенной SIM
